### App Settings

**What is an API key and where do I get it?**
Simply put: An API key is like a pass that allows your app to use the smart assistant. Without this pass, the assistant won't be able to answer questions.

**Where to get this "pass"?**
1.  **Open the site:** Go to Google AI Studio.
2.  **Log in:** Use your regular Google account (the same one for Gmail or YouTube).
3.  **Create a key:** Click the "Create API key" button.
4.  **Copy it:** A long string of letters and numbers will appear ‚Äì copy it.
5.  **Paste into the app:**
    *   Open settings (the gear icon in the top right corner).
    *   Paste the copied string into the "API key" field.
    *   Click "Save".

**Important:** This key is FREE! Google gives it to everyone. You can ask many questions every day without payment.

**What is "Model Selection" in settings?**
Simply put: It is like choosing a version of the smart assistant ‚Äì faster or smarter.

**What are the options:**
üü¢ **Gemini 2.5 Flash Lite** (selected by default)
*   Responds very quickly.
*   Suitable for most questions.
*   You can ask many questions in a row.

üü° **Gemini 2.0 Flash**
*   Slightly smarter than the first option.

üîµ **Gemini 2.0 Flash Lite Preview**

**When should you change the model?**
*   If you see a "too many requests" error ‚Äì try a different model.

**Language Switching (EN/RU)**
The button in the top right corner allows you to switch between:

**Important:** You can ask questions in any language regardless of this setting! It only affects which translation of the books is shown to you.
